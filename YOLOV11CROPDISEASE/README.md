# 🌾 农作物病虫害智能检测系统

基于 YOLOv11 的农作物病虫害智能识别 Web 应用系统，支持 38 种作物病害分类检测，提供专业的防治建议。

## ✨ 功能特性

- 🔐 **用户认证** - 支持用户登录/登出，区分管理员和普通用户角色
- 🖼️ **图片检测** - 支持单张/批量图片上传检测
- 📹 **视频检测** - 实时视频帧捕获和检测
- 📊 **历史记录** - 检测记录持久化存储，支持环境数据记录
- 💊 **防治建议** - 根据病害类型和严重程度提供专业防治方案
- 🗂️ **图片管理** - 自动清理过期图片，支持手动管理

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 后端框架 | Flask 2.3.3 |
| AI 模型 | YOLOv11 (Ultralytics 8.4.21) |
| 深度学习 | PyTorch |
| 数据库 | SQLite |
| 图像处理 | Pillow 10.1.0 |
| 前端 | HTML5 / CSS3 / JavaScript |

## 🚀 快速开始

### 1. 环境要求

- Python 3.8+
- pip 包管理器

### 2. 安装依赖

```bash
cd APP
pip install -r requirements.txt
```

### 3. 运行应用

```bash
python app.py
```

应用将在 `http://localhost:5000` 启动

### 4. 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |
| user | user123 | 普通用户 |

## 📖 使用说明

### 图片检测

1. 登录系统后进入检测主页面
2. 点击"选择文件"上传图片（支持 jpg/jpeg/png 格式）
3. 点击"开始检测"执行 AI 检测
4. 查看检测结果和防治建议

### 批量检测

- 支持同时选择多张图片上传
- 系统自动并发处理，提高检测效率

### 视频检测

1. 进入视频检测页面
2. 允许浏览器访问摄像头
3. 实时捕获视频帧进行检测

### 历史记录

- 查看所有检测记录
- 支持添加环境数据（温度/湿度/降雨量）
- 管理员可查看所有用户记录

## 🎯 病害等级判定

系统根据检测置信度自动判定病害严重程度：

| 等级 | 置信度范围 |
|------|-----------|
| 重度 | > 80% |
| 中度 | 50% - 80% |
| 轻度 | 30% - 50% |
| 轻微 | < 30% |

## 📝 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET/POST | 登录页面 |
| `/index` | GET | 检测主页面 |
| `/detect` | POST | 图片检测接口 |
| `/history` | GET | 历史记录页面 |
| `/add_env` | POST | 添加环境数据 |
| `/delete_record/<id>` | GET | 删除单条记录 |
| `/video_detect` | GET | 视频检测页面 |
| `/detect_video_frame` | POST | 视频帧检测接口 |
| `/manage_images` | GET | 图片管理页面 |
| `/logout` | GET | 退出登录 |
